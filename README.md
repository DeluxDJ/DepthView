# DepthView
虛擬貨幣深度圖<br>

#導入專案<br>
## Add it in your root build.gradle at the end of repositories: <br>

```
allprojects {
  repositories {
	...
	maven { url 'https://jitpack.io' }
  }
}
```
  <br>
  <br>
  
  
## Add the dependency <br>
```
dependencies {
       implementation 'com.github.billuser:DepthView:v1.3'<br>
}
```
